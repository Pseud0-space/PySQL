# PySQL
A front end for MySQL based on python3.

1. A file named as "MySQL_Password.txt" containing the MySQL root password
2. A file named as "MySQL_DB.txt" containing the name of MySQL Database used

# Python imports - 

- PyQt5
- mysql-connector
- cryptography
- base64
- os
- sys
